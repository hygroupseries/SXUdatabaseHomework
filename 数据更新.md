将数据库这门课的学分数修改为5

```
UPDATE course
SET ccredit = 5
WHERE cname = '数据库';
```

删除选修了C1课程的所有选课记录 

```
DELETE FROM sc
WHERE cno = 218001;


```

将软件学院选修了C2课程的学生的该门成绩增加10分

```
UPDATE sc
JOIN student ON sc.sno = student.sno
SET sc.grade = sc.grade + 10
WHERE student.smajor = '软件工程' and sc.cno = 218002;

UPDATE sc
SET grade = grade + 10
FROM student
WHERE sc.sno = student.sno 
AND student.smajor = '软件工程' 
AND sc.cno = 218002;
```
